- tải helper đi kèm: $this->load->helper('form'); - s/d hàm : $this->form_validation->hàm(); * Các phương thức : - set_rules : tạo các tập luật , 3 tham số : + vtri 1 : name form + vtri2 : name xuất ra tb lỗi + vtri 3: tập luật cách nhau "|". vd : $this->form_validation->set_rules('name', 'họ tên', required|min_length[8]'); ++ các tập luật hay dùng : + required : y/c nhập ko đc để trống; + matches: s/d khi 2 ô nhập dữ liệu giống nhau (nhập lại pass); + min_length : số ký tự tối thiểu + max_length : số ký tự tối đa;; + numeric : y/c trong textbox phải nhập số; +valid_email : email bắt buộc đúng định dạng; + xss+clean : xóa xss của input, bảo mật; + exact_length: ktra đ/k bằng độ dài đc gắn gtri (exact_lenght[8]) + greater_than : ktra đ/k phải lớn hơn độ dài đc gắn giá trị(greater_than[8]) + les_than : ktra đ/k phải nhỏ hươn độ dài đc gắn gtri(les_than[8]) +callback_ : gọi 1 hàm xử lý khác - hàm run(): ktra các tập lệnh trên hợp lệ hay ko kết quả trả về : true/false;
